# open-kattis-solved-c
My solutions with C to coding problems of open.kattis.com 

## Building
```
gcc -Wall -g program.c -o program.out
```

## Running
time ./program.out < sample.txt > output.txt

example output:
```
real    0m0.003s
user    0m0.002s
sys     0m0.002s
```

## Validating output
diff output.txt correct.txt
